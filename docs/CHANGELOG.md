# Changelog

## [1.0.0] - 2025-01-09
### Added
- Email scanning functionality to analyze sender, recipient, and email content.
- Suspicion level calculation based on keywords.
- Email history tracking (logging details of processed emails).
- Basic user interface for manual email entry (via console).

## [1.1.0] - 2025-01-15
### Added
- Keyword weights configuration via `keyword_weights.json`.
- Option to modify and update keyword weights.
- Error handling for invalid email formats and empty inputs.

## [1.2.0] - 2025-01-25
### Added
- Graphical User Interface (GUI) support (implementation ongoing).
- Enhanced email content preprocessing (removed punctuation, case normalization).
